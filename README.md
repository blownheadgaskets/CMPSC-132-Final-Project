# CMPSC-132-Final-Project

## Project decription
- This is a terminal based number guessing game that has the user attempt to guess an integer from 1 to 100 in ten attempts
- There are four enhancements beyond the base functionality that is requred. They are difficulty levels, limited attempts,
guess history for the hard level, and replay 
- The medium difficulty level is essentially the base functionality, where the user is prompted if they are too high or too low
- The hard and easy difficulty levels have different custom functionality

## Instructions to run the program on windows:
- Open the file in a text editor such as vscode
- If using vscode you can press the run button in the top right corner to run the file in the built in terminal. Otherwise, if you are using a regular terminal and you are in the right directory, you can run the file by typing python followed by the file name (e.g. python "Final Project.py")
- Once the file is running follow the prompts to the user. The first input should be the enter key when you start the game.
